Automacorp

Welcome on this Automacorp project made by Dev-Mind. This application is an example on how you can implement security in your app. You can explore this api with Swagger

This API is secured with Spring security, but you have 2 users for tests

  • a user called `user` with a password equals to `password`
  • a super user called `admin` with a password equals to `password`

The H2 console is only available to super users
